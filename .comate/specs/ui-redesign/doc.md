# 极海Channel主页UI重设计 - 需求规格文档

## 1. 需求概述

重新设计极海Channel主页UI，使其交互更友好、视觉更简约。

## 2. 当前问题分析

### 2.1 视觉问题
- 渐变、发光、阴影效果过多，视觉噪音大
- 配色过于花哨（紫色、青色、粉色多种渐变）
- 卡片边框和背景层次过多

### 2.2 交互问题
- 服务流程步骤展示过于详细，信息密度过高
- 两个"立即预约"按钮功能相同但分散注意力
- 页面内容过长，用户需要滚动才能看到全部信息

### 2.3 结构问题
- bitfree社区和个性化服务两个模块视觉权重相近，主次不够分明
- 支付方式图片展示占用过多空间

## 3. 设计目标

1. **极简视觉** - 减少颜色种类，使用更克制的配色方案
2. **信息聚焦** - 突出核心内容，减少次要信息的视觉干扰
3. **交互简化** - 减少用户决策成本，优化操作流程
4. **响应友好** - 确保移动端和桌面端都有良好体验

## 4. 设计方案

### 4.1 配色方案

采用深色主题，但大幅简化配色：

```css
:root {
  --bg-primary: #0d1117;      /* 主背景 - 更纯净的深色 */
  --bg-secondary: #161b22;    /* 次级背景 */
  --bg-card: #21262d;         /* 卡片背景 */
  --bg-hover: #30363d;        /* 悬停背景 */
  
  --text-primary: #f0f6fc;    /* 主文字 */
  --text-secondary: #8b949e;  /* 次级文字 */
  --text-muted: #6e7681;      /* 弱化文字 */
  
  --accent: #58a6ff;          /* 单一强调色 - 科技蓝 */
  --accent-hover: #79c0ff;    /* 强调色悬停 */
  --border: #30363d;          /* 边框色 */
}
```

### 4.2 布局重构

#### Header区域（侧边栏）
- 保持现有结构，简化样式
- 移除过多的发光效果
- 社交链接使用更简洁的图标样式

#### 主内容区域

**Hero区域 - 个人简介**
- 添加简洁的个人介绍
- 一句话描述核心价值主张

**bitfree社区推广**
- 简化为单行卡片设计
- 左侧Logo/图标 + 中间描述 + 右侧按钮
- 移除渐变背景，使用纯色或极淡背景

**服务区域 - 选项卡设计**
- 将两个服务合并为选项卡切换
- 默认显示"模拟面试"（热门）
- 点击切换查看"职业咨询"
- 大幅简化流程展示，只显示3个关键步骤

**联系方式**
- 简化支付二维码展示
- 使用更紧凑的布局

### 4.3 组件设计

#### 选项卡组件
```
[ 模拟面试 ] [ 职业咨询 ]
-------------------------
| 服务描述               |
| 步骤1 → 步骤2 → 步骤3 |
| [立即预约]             |
-------------------------
```

#### 按钮样式
- 使用单一蓝色作为主按钮
- 移除渐变和发光效果
- 悬停时仅改变背景色和轻微位移

#### 卡片样式
- 使用1px细边框
- 移除阴影和发光效果
- 悬停时边框颜色变化

### 4.4 交互优化

1. **选项卡切换** - 点击切换服务类型，减少页面长度
2. **平滑滚动** - 添加锚点平滑滚动
3. **悬停反馈** - 所有可交互元素都有清晰的悬停状态
4. **加载动画** - 页面元素渐入动画，但更加克制

## 5. 受影响文件

| 文件 | 修改类型 | 说明 |
|------|----------|------|
| `/_layouts/default.html` | 修改 | 简化CSS样式，重构HTML结构 |
| `/index.md` | 修改 | 重构内容结构，添加选项卡HTML |

## 6. 实现细节

### 6.1 CSS变量重新定义

完全替换现有的复杂CSS变量为简约版本。

### 6.2 响应式断点

- Desktop: > 800px - 侧边栏布局
- Tablet: 600px - 800px - 侧边栏变顶部
- Mobile: < 600px - 单列布局，选项卡全宽

### 6.3 性能考虑

- 减少CSS复杂度
- 移除不必要的动画
- 保持图片懒加载

## 7. 预期效果

1. **视觉** - 更专业、更现代的深色主题设计
2. **交互** - 用户可以在不滚动的情况下了解所有服务
3. **转化** - 更清晰的CTA按钮和简化流程可能提升预约转化率
4. **体验** - 更快的页面加载和更流畅的交互
